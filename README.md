# دليل رفيق العبادة الذكي (Worship Companion PWA)

تطبيق ويب تقدمي (PWA) يعمل كرفيق عبادة يومي ذكي وتفاعلي يركز على تكييف الواجهة وعرض المحتوى ديناميكياً بناءً على سياق المستخدم الحالي (Current Context) بدلاً من عرض القوائم والأقسام الثابتة التقليدية.

---

## 🌟 الفلسفة الأساسية للتطبيق

التطبيق ليس مجرد أرشيف للكتب أو الأدعية؛ بل هو **رفيق يومي نشط**. تتغير الصفحة الرئيسية ديناميكياً لتلائم:
- **الوقت الحالي** (فجر، ضهر، عصر، مغرب، ليل، إلخ).
- **الصلاة القادمة** والوقت المتبقي لها.
- **اليوم والليلة الهجرية والملقاة** (مثلاً ليلة الجمعة أو نهار الجمعة).
- **المناسبات الإسلامية الهجرية الكبرى** (مثل عاشوراء، المولد النبوي، إلخ) وتأثيرها على ترتيب المحتوى والأدعية المستحبة.

---

## ⚙️ محرك السياق الذكي (Smart Context Engine)

تم استبدال الشروط البرمجية الصلبة (مثل `if(prayer === 'fajr')`) في واجهة المستخدم بمحرك ذكي مركزي متكامل يقع في `/src/utils/contextEngine.ts`.

### 1. كائن السياق الحالي (`CurrentContext`)
في كل دقيقة أو عند فتح التطبيق، يقوم المحرك ببناء كائن سياق داخلي يمثل الحالة الزمنية والدينية الراهنة للمستخدم بدقة:

```typescript
interface CurrentContext {
  currentTime: Date;               // الوقت الحالي بالتفصيل
  currentPrayer: 'fajr' | 'dhuhr' | 'asr' | 'maghrib' | 'isha' | 'none';
  targetPrayer: 'fajr' | 'dhuhr' | 'asr' | 'maghrib' | 'isha';
  phase: 'before_prayer' | 'during_prayer' | 'after_prayer' | 'normal';
  minutesToNext: number;           // الدقائق المتبقية للأذان القادم
  minutesSinceLast: number;        // الدقائق المنقضية منذ الأذان الماضي
  weekday: string;                 // اليوم الحالي (عربي)
  isIslamicNight: boolean;         // هل الوقت يقع في الليلة الإسلامية (تبدأ من المغرب)
  nightOf: string;                 // الليلة الحالية (مثال: "ليلة الجمعة")
  hijriDay: number;                // اليوم الهجري
  hijriMonth: number;              // الشهر الهجري الحالي
  hijriMonthName: string;          // اسم الشهر الهجري بالعربية
  events: string[];                // المناسبات الدينية النشطة في هذا اليوم
}
```

### 2. تغذية الصفحة الرئيسية الديناميكية (`Smart Feed`)
بدلاً من عرض بطاقات ثابتة، يتم الاستعلام من جدول `smart_feed` المحلي في قاعدة البيانات (نسخة Dexie v2) الذي يحتوي على قواعد التفعيل والأولويات:

```typescript
interface SmartFeedItem {
  id: string;
  trigger_type: 'prayer_phase' | 'weekday' | 'hijri_date' | 'event' | 'always';
  trigger_value: string;           // مثل "fajr:before_prayer", "friday", "10-01"
  priority: number;                // أولوية العرض (العناصر الأكبر تظهر أولاً)
  contentType: 'hadith' | 'wisdom' | 'dua' | 'ziyarat' | 'taqiba' | 'event' | 'munajat' | 'custom';
  contentId?: string;              // معرف النص الفريد في الجداول الأخرى
  title: string;                   // العنوان المعروض
  description?: string;            // شرح مختصر للعمل
  actions?: string[];              // قائمة مهام تفاعلية (Checklists) قابلة للتعليم والتقدم
}
```

يقوم المحرك بمطابقة السياق الحالي مع قواعد جدول `smart_feed` لتوليد تيار زمني (Timeline) تفاعلي متناسق ومرتب تنازلياً حسب الأولويات.

---

## 🗄️ بنية البيانات المحلية ومزامنتها (Offline-First)

يتم تخزين جميع البيانات في المتصفح محلياً باستخدام **Dexie.js (IndexedDB)** لضمان سرعة فائقة جداً وعمل التطبيق 100% بدون إنترنت بعد التشغيل الأول.

### ترقية قاعدة البيانات (Schema Update v2)
تمت ترقية قاعدة بيانات `WorshipDatabase` إلى الإصدار الثاني لإدخال جدول تفعيل التغذية الذكية:
- **`smart_feed`**: `id, trigger_type, trigger_value, priority`

### نظام الاستيراد الذاتي للبيانات (`importer.ts`)
- تتم قراءة هيكلية ملفات الـ JSON الفارغة أو المبدئية ومزامنتها تلقائياً من المجلد `/public/data/` عند توفر الإنترنت.
- يتتبع التطبيق إصدارات المحتوى محلياً عبر `content_versions.json` لضمان تحديث القواعد العبادية في الخلفية دون الحاجة لإعادة تحميل التطبيق أو تحديث الكود المصدري.

---

## 🎨 الواجهة والتجربة البصرية (UI & Motion)

تمت إعادة تصميم الواجهة بالكامل لتتحول إلى **شريط زمني تفاعلي (Smart Timeline)**:
- **تأثيرات حركية فائقة السلاسة (`motion/react`)**: استعراض متدرج staggered للبطاقات عند دخول الصفحة أو تغير السياق.
- **قوائم المهام المستحبة التفاعلية**: يمكن للمستخدم وضع علامة صح على الأذكار والتعقيبات والخطوات المنجزة وتتبع تقدمه لليوم الحالي.
- **توجيه وقت الصلاة**: عند دخول وقت الفريضة، تبرز الواجهة وتوفر زراً فورياً للانتقال لبوصلة وحساب اتجاه القبلة.
- **أوضاع العرض المريحة للعين**: تدعم الواجهة الوضع الليلي المتوافق والدافئ والألوان المريحة المناسبة لأوقات السحر والمناجاة في الساعات المتأخرة.
